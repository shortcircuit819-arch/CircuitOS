# CircuitOS v0.4.3 — Overlay Labels & Background Fix

## What's New

### Background image now applies to the panel

The background image field now applies the image directly to the lower-third
panel card rather than to the browser source body. The image sits behind the
panel's gradient and opacity layers — set Opacity lower to let the image show
through more strongly. The field accepts any `https://` URL or a local
`file:///C:/path/to/image.png` path.

### Editable overlay labels

A new **Labels** section in the Overlay Editor lets you change every static
text string on the overlay:

| Field | Default |
|-------|---------|
| Scan label | CIRCUIT SCAN |
| Component label | COMPONENT ACQUIRED |
| Progress label | COLLECTION PROGRESS |
| New item badge | NEW COMPONENT |
| Complete badge | COLLECTION COMPLETE |
| Duplicate badge | DUPLICATE |

Text is automatically uppercased. The rare pull badge uses the label from the
Streamer.bot action and cannot be changed here.

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
   - `Overlay\` folder
3. Restart CircuitOS and open **Settings → Overlay Editor**.
