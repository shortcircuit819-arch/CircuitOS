# CircuitOS 0.9.0 — Distribution & Release Candidate

0.9 is about how CircuitOS gets to you and keeps itself current. This is the release candidate for 1.0:
a proper installer, in-app updates, and a signed build pipeline.

## A real installer

CircuitOS now ships as a **Setup.exe** instead of a ZIP you unpack yourself. It installs per-user (no
admin prompt), makes Desktop and Start Menu shortcuts, and is what makes automatic updates possible.

Already running the ZIP version? Install the setup once and you're on the updater from then on. **Your
data is untouched** — collections, inventories, and profiles live in `Documents\CircuitOS\Data`,
completely separate from the program. Updates only ever swap program files.

## Check for updates, in the app

**Settings → About** has a **Check for updates** button. When a newer version exists you'll see
`0.9.0 → 0.9.1` and a **Download & Restart** button that fetches the update and relaunches into it.

Installed from a ZIP or running a dev copy? The panel tells you so rather than pretending — those copies
update by running the new Setup.exe once.

## Signed releases

The release pipeline now signs the installer and the app, with timestamping, in one command. Signing is
what removes the "Windows protected your PC — unknown publisher" warning that makes a new app feel
sketchy.

## Notes

- Nothing in your data changes. Local mode is still the default.
- Automatic updates require the release feed to be publicly reachable — if updates report a fetch error,
  that's the feed, not your install.
