# CircuitOS v0.4.5 — Working Overlay Background & Image Upload

## What's New

### Overlay background image — working

The background image now correctly fills the full OBS Browser Source canvas.

**Root cause of previous failures:** The CSS shorthand `html, body { background: transparent }` was resetting `background-image: none`, overriding the `--bg-image` CSS variable. Additionally, earlier versions placed the image inside the `.tracker` panel background stack where it was hidden behind near-opaque (`0.98`) gradients.

**Fix:** The `html` rule now only sets `background-color: transparent`. The `body` rule sets `background-image: var(--bg-image, none)` with `cover/center` — body fills the full OBS canvas and the tracker panel sits on top.

### Upload Image button

The Overlay Editor now has an **Upload Image** button. Click it to pick a local file (PNG, JPG, GIF, WEBP) — the image is sent to CircuitOS and served directly from `GET /overlay-bg`, so no file path configuration is needed.

### Larger overlay preview

The overlay preview iframe in the Overlay Editor has been taller (300 → 500 px) to give a better view of the lower-third panel.

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
   - `Overlay\` folder
3. Restart CircuitOS and open **Settings → Overlay Editor**.
4. Click **Upload Image** to set a background, or leave it blank for a solid dark panel.
