# CircuitOS v0.5.0.6 — Overlay Customization & Live Preview

## What's New

### New overlay customization controls

The Overlay Editor now exposes controls that were previously hardcoded, giving
you full control over the visual style of every element on screen.

**Colors (Appearance section)**
- **Label color** — controls the eyebrow text ("CIRCUIT SCAN"), component label
  ("COMPONENT ACQUIRED"), and the status badge (e.g. "NEW COMPONENT") in their
  default state. Previously this was always tied to the red accent and could not
  be changed independently.
- **Bar fill color** — sets the progress bar fill. Previously hardcoded red.

**Sizes (Display & Animation section)**
- **Viewer name (px)** — font size for the viewer name. Default 38 px,
  range 16–72 px.
- **Item name (px)** — font size for the component/item name. Default 34 px,
  range 14–64 px.
- **Label size (px)** — font size for the small uppercase kicker labels.
  Default 14 px, range 8–24 px.

**Layout section**
- **Bar height (px)** — height of the progress bar track. Default 8 px,
  range 2–32 px.

### Live preview

Controls in the Overlay Editor now update the preview panel immediately as you
change them — no save required to see the result. The Save button still commits
changes to disk.

### Preview state switcher

The preview panel header now has four buttons — **Normal**, **Rare**,
**Complete**, **Duplicate** — so you can inspect every badge state without
waiting for a real pull.

### Preview no longer blank on a fresh profile

The preview previously showed a black panel when no pull had ever been recorded.
It now always renders a placeholder pull in preview mode so the overlay is
visible from first use.

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
3. Restart CircuitOS.
