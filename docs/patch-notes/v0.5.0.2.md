# CircuitOS v0.5.0.2 — Nav & Label Polish

## What's New

### Cleaner sidebar navigation

The sidebar has been reorganized to match the natural first-run flow.

**Configure** (previously "Settings") is now positioned right after Profiles — the first
place a new user needs to go after the wizard. It contains Game Profile, Overlay Editor,
and Messages.

**Catalog** (previously "Collections") now correctly describes the group it contains —
Collections, Events, and Featured Boost are all catalog management tools, and the old
"Collections" group name collided with the "Collections" item inside it.

### "Game Profile" replaces "Branding"

The "Branding" nav item and panel heading are now called **Game Profile**. The section
controls your game's name, terminology, commands, colors, and chat messages — "Branding"
undersold what it does.

### "Circuit Components Starter" in the first-run wizard

The starter preset was named "Circuit Components" — the same as the default game name —
which made it read like the app name rather than a template. It is now **Circuit Components
Starter** to make clear it is an example you can customize, not the name of the software.

### Label clarity improvements

- **Dupe protection (pulls)** — was "Dup protection (turns)". Spelled out and uses "pulls"
  to match the game's own vocabulary. Tooltip now fully explains the guarantee window.
- **Export Active Profile** — was "Export Active" on the Profiles toolbar. The previous
  label was ambiguous about what was being exported.

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
3. Restart CircuitOS.
