# CircuitOS v0.5.0.1 — Admin UI Cleanup

## What's New

### Clearer button labels

Several buttons in the admin panel have been renamed to reduce ambiguity:

- **Save Live Config** → **Save Catalog** — makes it clear this saves the collection catalog and boost config, not the system profile or overlay
- **Import** / **Export** (top bar) → **Import Catalog** / **Export Catalog** — distinguishes catalog import/export from the profile module import/export on the Profiles page

### Streamer.bot directly in the sidebar

The "Integrations" collapsible group has been removed. Streamer.bot now appears as a direct nav item — one fewer click to get there.

### Consistent simulator naming

The nav entry previously labeled **Pull Lab** now reads **Simulator**, matching the view heading.

### Eyebrow label field renamed

The **Brand kicker** field (in the first-run wizard and Branding view) is now labeled **Eyebrow label** with a tooltip explaining what it does. The field behavior is unchanged — this is a display name only.

---

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
3. Restart CircuitOS.

No data migration required.
