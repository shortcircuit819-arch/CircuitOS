# CircuitOS v0.4.4 — Overlay Background (Attempt 2)

## What's New

### Overlay background image — revised approach

A second iteration of background image support for the lower-third overlay.
This version moved the image application earlier in the render pipeline and
adjusted the CSS layering. The image field in the Overlay Editor accepts any
`https://` URL or local `file:///` path.

> **Note:** Background image display was not yet fully resolved in this
> release — the image was still being hidden behind the panel's near-opaque
> gradient layer. The root-level body fix landed in 0.4.5.

---

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
   - `Overlay\` folder
3. Restart CircuitOS.
