# CircuitOS v0.4.0 — Overlay Editor

## What's New

**Overlay Editor** is now built into the admin panel under the new "Overlay Editor"
nav item. Configure the OBS lower-third without touching any files.

### Lower-Third Controls

- **Layout** — position (bottom-center / bottom-left / bottom-right / top-left /
  top-right), width, minimum height, horizontal and vertical offset
- **Colors** — background, panel, accent, text, muted, and background opacity
- **Timing** — display duration in seconds, enter/exit transition lengths in
  milliseconds, animation style (slide / fade / none), and font family
- **Content toggles** — enable or disable the overlay entirely, collection name,
  progress bar, and CircuitOS branding

### Live Preview

The editor embeds the actual OBS lower-third in a scaled preview pane using
the last recorded pull. Click **Save Overlay Config** to write the changes to
disk and refresh the preview. Click **Refresh Preview** to reload without saving.

### Overlay File Serving

The runtime now serves the overlay files over HTTP so they can be embedded in
the admin panel and used as a direct OBS Browser Source URL:

```
http://127.0.0.1:8787/overlay/index.html
```

---

## How to Update (Existing Install)

1. Close CircuitOS if it is running.
2. Open your install folder and copy these files from the update package, replacing
   the old ones:
   - `CircuitOS.exe`
   - `App\` folder (all files)
3. Restart CircuitOS — your data and inventory are unchanged.
4. Click **Overlay Editor** in the sidebar to try the new editor.
