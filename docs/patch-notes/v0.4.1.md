# CircuitOS v0.4.1 — Overlay Editor Polish

## What's New

### Configurable Redemption Cooldown

A per-viewer cooldown is now built into the Streamer.bot action.

- Set the cooldown in seconds under **Settings → Branding** (field: *Redeem cooldown (sec)*).
  Default is 120 seconds (2 minutes). Set to 0 to disable.
- When a viewer redeems before their cooldown expires, the redemption is
  automatically refunded and a chat message tells them how many seconds remain.
- The cooldown value is injected into the generated action when you click
  **Regenerate Actions** on the Integrations tab — regenerate after changing it.

### Custom Background Image

The Overlay Editor now includes a **Background image URL** field under Appearance.
Paste any local file path or web URL to set a background image on the OBS
lower-third. Combine with the Opacity slider to blend it with the overlay colors.

### Overlay Editor Fixes

- **Preview no longer constantly refreshes.** The overlay's 500 ms state-polling
  interval is now disabled when running in preview mode.
- **Save and Refresh now reliably reload the preview.** A timestamp cache-buster
  is appended to the iframe URL so the browser always fetches a fresh copy.

### Sidebar Reorganization

The left navigation is now grouped and ordered by workflow:

- **Overview** — dashboard (top-level)
- **Collections** — Collections, Events, Featured Boost
- **Overlay Editor** — lower-third editor (top-level)
- **Viewers** — Inspector, Discord Roles, Economy
- **Settings** — Branding, Messages
- **Integrations** — Streamer.bot setup
- **Tools** — Pull Lab, Patch Notes, Backups

## How to Update (Existing Install)

1. Close CircuitOS if it is running.
2. Copy these files from the update package into your install folder, replacing
   the old ones:
   - `CircuitOS.exe`
   - `App\` folder (all files)
3. Restart CircuitOS.
4. Go to **Settings → Branding** to set your cooldown, then click
   **Regenerate Actions** under **Integrations → Streamer.bot** to push the
   updated action to Streamer.bot.
