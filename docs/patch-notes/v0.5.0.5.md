# CircuitOS v0.5.0.5 — Inventory Admin & Import UX

## What's New

### Viewer inventory admin actions

The Viewer Inventory inspector now includes admin controls for correcting data
without leaving the panel.

- **Reset Inventory** — removes a viewer's entire item history. Use this to give
  a viewer a clean slate. A timestamped backup is created before the change is
  written, and a confirmation prompt appears first.
- **Remove item** — hover any owned item in a viewer's collection to reveal a ×
  button. Clicking it removes that specific item from the viewer's inventory with
  the same backup-then-write safety guarantee.

Both actions reload the analytics panel automatically when complete.

### Viewer list cleanup

- Display names only — Twitch IDs and scrap balances are no longer shown in the
  viewer list sidebar. The ID is still used for search; it just isn't displayed.
- The "READ ONLY" chip in the viewer detail header has been replaced by the
  **Reset Inventory** button.

### Import error UX

When an import has row-level errors, the preview now clearly separates what's
wrong from what's ready:

- A compact error summary at the top lists the affected items and their specific
  error (up to 10 shown; remaining errors counted).
- The preview table shows **only the ready items** so you don't scroll through
  thousands of rows hunting for problems.
- A **Skip Errors — Apply N** button lets you apply all ready items immediately
  without fixing the source data first. The original Apply button remains and
  requires all errors to be resolved.

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
3. Restart CircuitOS.
