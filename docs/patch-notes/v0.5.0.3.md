# CircuitOS v0.5.0.3 — Sidebar Overhaul

## What's New

### Inline profile switcher

The active profile block in the sidebar is now a clickable switcher. Click it to open a
dropdown showing all your profiles — the active one is highlighted. Click any other profile
to switch directly without navigating away from whatever view you're in. The list scrolls
if you have more profiles than fit.

A **Manage profiles** link at the bottom of the dropdown takes you to the full Profiles
view for creating, renaming, and deleting profiles. The dropdown closes on outside click
or Escape.

### "Community" nav group

The **Viewers** group has been renamed to **Community** and reorganized:

- **Inventory** (was "Inspector") — viewer item collections
- **Discord Roles** — role award queue and history
- **Economy** — scrap balances and duplicate pressure
- **Patch Notes** — moved here from Tools; it's community communication, not a dev tool

### Sidebar cleanup

- The "CIRCUITOS PLATFORM" label above the CircuitOS logo has been removed — the name
  says what it is.
- "Profiles" has been removed from the nav — the profile switcher above handles switching,
  and "Manage profiles" in the dropdown reaches the full view.
- "CIRCUITOS LOCAL ENGINE" has been removed from the footer — the connection dot, version,
  and data path are self-explanatory.
- Nav group expand/collapse indicators are now chevrons instead of +/− symbols.

---

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
3. Restart CircuitOS.
