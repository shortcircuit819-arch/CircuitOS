# CircuitOS v0.4.6 — Patch Notes Builder, Diagnostics & Dup Shield

## What's New

### Patch Note Builder

A new **Patch Notes** section (under Tools) generates a Discord-ready update
post from your actual catalog changes. Open it after editing your collections
and it auto-detects added, renamed, and removed items. Add an opening message
and any extra notes, then copy the formatted text directly into Discord.

Use **Mark Published** after posting to reset the comparison point for the
next cycle.

### Privacy-safe Diagnostics Download

**System Overview → Configuration Health** now has a **Download Diagnostics**
button. It produces a JSON file with your collection structure, item counts,
validation results, and pull rates — no viewer inventory or personal data
included. Useful for sharing with support or troubleshooting configuration
issues.

### Duplicate protection (turns)

A new **Dup protection (turns)** field in **Settings → Branding** lets you
configure a guaranteed-new-item window after a viewer pulls a duplicate.
Setting this to `N` means the next `N` pulls for that viewer will be forced
to land on items they don't own yet.

After changing this setting, regenerate and reinstall the Streamer.bot actions
from the **Streamer.bot** tab.

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
   - `Overlay\` folder
   - `Streamerbot Actions\` folder (if using dup protection)
3. Restart CircuitOS.
4. If enabling dup protection: go to **Streamer.bot**, regenerate C#, and
   replace the existing Execute C# sub-action in Streamer.bot.
