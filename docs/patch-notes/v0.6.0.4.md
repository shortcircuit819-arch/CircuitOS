# CircuitOS 0.6.0.4 — Rate Lab Tier Breakdown & Tier-Aware Simulation

## What's New

### Rate Lab — Rarity Tiers breakdown panel

The "COMING IN 0.6" placeholder in Rate Lab is now live. The **Rarity Tiers** panel
shows a breakdown for every active collection that has tiers configured:

- **Tier label** — the name you gave this tier (e.g., COMMON, RARE, ULTRA RARE)
- **Item count** — how many items are assigned to this tier
- **% of all pulls** — the effective probability that a pull from this collection
  lands in this tier, expressed as a share of *all* pulls across all collections
- **Weight bar** — visual proportion of this tier relative to the collection
- **Per-item odds** — approximate 1-in-N odds for any single item in this tier

Collections without tiers are not shown. If no active collections have tiers, the
panel shows a helpful empty state.

### Tier-aware pull simulation

The pull simulator now accounts for tier weights when generating its results.
Items in heavier tiers appear more often in simulations; items in lighter tiers
appear less often — matching the live weighted odds exactly.

Previously, the simulator assumed all items within a collection pulled at equal odds.
If you run simulations on a collection with tiers, re-run them to see the corrected
distribution.
