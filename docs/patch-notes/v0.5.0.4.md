# CircuitOS v0.5.0.4 — Rate Lab

## What's New

### Rate Lab replaces the Simulator

The **Simulator** view has been rebuilt into the **Rate Lab** — a unified workspace for
tuning collection weights and verifying pull distribution.

#### Weight editor

All collection weights are now editable in a single table. Each row shows:

- The collection name (with an active/inactive badge for events)
- An editable weight input
- A live percentage bar that recalculates as you type
- The exact percentage share

Weight changes feed directly into the main **Save Catalog** flow — no separate save
button. The weight editor is divided into **Permanent** and **Events** sections.

#### Distribution check

Click **Run Check** to simulate the configured number of redeems and compare the
random distribution against the exact pull model. The results reset automatically
whenever you change a weight, so you always know whether the simulation reflects the
current settings.

#### Rarity Tiers (placeholder)

A **Rarity Tiers** panel is shown at the bottom of Rate Lab. Tiers are a 0.6 feature
that will let you assign items within a collection to named rarity levels (Common,
Rare, Ultra Rare, or whatever names you choose) with independent pull weights. Items
without tiers always pull at equal odds within their collection.

### Overview panel updates

- The **WEIGHT MODEL** panel on the Overview is now labeled **PULL ODDS**.
- The heading changes dynamically: **Pull Rates** at base rates, **Boosted Pull Rates**
  when a featured boost is active.
- The weight total chip is replaced by a **BASE RATES** / **BOOST ACTIVE** indicator.
- A **Tune in Rate Lab** button links directly to the Rate Lab from the Overview.

### `?` tooltip helpers

Small `?` buttons have been added to technical UI sections. Hover or focus them to
read a plain-English explanation of what the section does. Rate Lab introduces the
first two: one for how collection weights work, one for what rarity tiers are.

## How to Update (Existing Install)

1. Close CircuitOS.
2. Copy from the update package into your install folder:
   - `CircuitOS.exe`
   - `App\` folder
3. Restart CircuitOS.
