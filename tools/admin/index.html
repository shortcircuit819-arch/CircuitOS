How message templates work

Write the sentence exactly as viewers should see it. Placeholder chips insert live values such as the viewer name, item, progress, or balance.

Click a chip to insert it at the cursor. Hover or focus a chip to see what it means. Keep the braces intact. Native Twitch uses your changes right after you save; regenerate the Streamer.bot actions only if you use that path.
